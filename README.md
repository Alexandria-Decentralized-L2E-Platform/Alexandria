## Getting Started

### Prerequisites

- Yarn
  ```sh
  npm install -g yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Alexandria-Decentralized-L2E-Platform/Alexandria.git
   ```
2. Checkout dapp directory
   ```sh
   cd Alexandria/dapp
   ```
3. Install YARN packages
   ```sh
   yarn install
   ```

### Development

1. Checkout dapp directory
   ```sh
   cd Alexandria/dapp
   ```
2. Run yarn start command
   ```sh
   yarn start
   ```
